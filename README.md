# אפליקציית ניהול גבייה

אפליקציית ווב (פועלת בדפדפן, גם מהנייד) לניהול תהליך גבייה מול לקוחות, עם:
- רשימת גבייה לכל לקוח (משימה, תאריך מעקב, פרטי לקוח, סכומים, חשבונית, הערות)
- חלוקה ללקוחות ("קומפון" וכו') — לכל לקוח נתונים נפרדים, ואפשר לקשר כל לקוח לקובץ Google Sheets משלו
- סנכרון הדדי אוטומטי מול Google Sheets — שינוי בטבלה מופיע באפליקציה ולהפך
- תזכורות: באנר עליון שמציג רשומות עם "תאריך מעקב" שהגיע או עבר
- שליחת תזכורת ללקוח בקליק אחד — פותח את WhatsApp או את תוכנת המייל עם הודעה מוכנה לאישור ושליחה

## איך מריצים

לחיצה כפולה על `start.bat` תפתח את שני השרתים (Server + Client) ואת הדפדפן בכתובת:

http://localhost:5173

(אפשר גם לגשת מהנייד באותה רשת WiFi דרך הכתובת "Network" שמוצגת בחלון ה-Client, למשל `http://192.168.x.x:5173`)

להרצה ידנית:
```
cd server && npm install && npm start     # שרת על פורט 4000
cd client && npm install && npm run dev   # ממשק על פורט 5173
```

## חיבור ל-Google Sheets (חד פעמי)

לוחצים על הכפתור "חיבור ל-Google Sheets" באפליקציה ועוקבים אחרי ההוראות שם:
1. יצירת פרויקט ב-Google Cloud Console + הפעלת Google Sheets API
2. יצירת OAuth client (סוג Desktop app), עם redirect URI: `http://localhost:4000/api/google/oauth2callback`
3. הדבקת תוכן קובץ ה-JSON של פרטי הגישה באפליקציה
4. לחיצה על "התחברות לחשבון Google" ואישור הגישה

לאחר החיבור החד פעמי, האפליקציה זוכרת אותו (קובץ `server/data/google-token.json`) ולא צריך לחזור על התהליך.

## הוספת לקוח חדש וקישור לטבלה שלו

1. "ניהול לקוחות" -> "הוספת לקוח"
2. שם הלקוח (לדוגמה: קומפון)
3. הדבקת קישור לקובץ Google Sheets שלו (לדוגמה אקסל הגבייה הקיים, אחרי שמעלים אותו ל-Google Drive וממירים ל-Sheets, או כל Sheet עם אותן עמודות)
4. בורר הלקוח העליון מאפשר לעבור בין הלקוחות; כל אחד עם הנתונים והסנכרון שלו

חשוב: כדי שהסנכרון יעבוד, הקובץ ב-Google Sheets צריך את אותן כותרות עמודה (בשורה הראשונה) כמו בתבנית המקורית:
משימה | תאריך מעקב | שם לקוח | נייד לקוח | כתובת מייל | סכום החוב | מס' חשבונית | יתרה לתשלום נכון ליום | אמצעי תשלום | הערות / שליחת הודעה/שליחת מייל

## סנכרון

- בלחיצה על "סנכרן עכשיו" מתבצע סנכרון מיידי
- כשיש חיבור פעיל לגוגל, מתבצע סנכרון אוטומטי ברקע כל דקה
- הוספה/עריכה של רשומה באפליקציה נשלחת מיד לטבלה (אם מחוברים), והוספה/עריכה בטבלה נכנסת לאפליקציה בסנכרון הבא
